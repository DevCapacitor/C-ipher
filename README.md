### Prerequisites

* gcc
* cmake

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/DevCapacitor/C-ippher.git
   ```
2. change directory and make script executable
   ```sh
   cd c-ippher
   chmod +x Release.sh
   ```
3. run executable script, all binary compiled in Build
   ```sh
   ./Release.sh
   cd Build/Release
   ```
